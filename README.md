# 安安的素描本聊天框

本项目是一个将你在一个文本输入框中的文字写到安安的素描本上的项目

## AI声明

本项目99%的代码由AI生成

## 部署

本项目只支持windows

本项目不提供字体文件和安安图片, 需要你自己想办法加进来, 分别命名为`font.ttf`和`base.png`.
如果分辨率不一样的安安图片, 需要修改`main.py`的第57和第58行, 定义文本框的大小.

依赖库安装: `pip install -r requirements.txt `

运行`main.py`即可开始监听回车, 按下回车会自动拦截按键, 生成图片后自动发送.

如果发送失败等可以尝试适当增大`main.py`第10行的`DELAY`
